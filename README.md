## Структура программы «Бухгалтерия»

```bash
Import.Module.Package
├── main.py                    # основной модуль для запуска программы
├── dirty_main.py              # модуль dirty_main.py
├── requirements.txt           # rich — библиотека для красивого вывода в терминал
└── application/
    ├── salary.py              # функция calculate_salary
    └── db/
        └── people.py          # функция get_employees
```
